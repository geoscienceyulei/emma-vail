Project Overview

This project turns a global climate dataset into a clean, professional time-series animation that’s easy to understand at a glance. Instead of treating the maps like exploratory “data dumps,” I designed each frame like a finished cartographic product, with a clear title (including the month), a readable legend, and subtle reference layers that give you geographic context without competing with the climate data. The result is a looping visual that makes the seasonal rhythm of global precipitation patterns feel obvious and intuitive.

A big focus was keeping the animation honest. I used a single, universal classification and color ramp across all 12 months so the colors mean the same thing every time, which prevents the viewer from being tricked by shifting symbology. The “World from Space” projection and globe styling help reinforce the idea that this is a planetary-scale pattern, but I also had to accept tradeoffs: any global projection and any set of class breaks will generalize some local detail. Overall, the goal was to balance style and clarity with accuracy, and to show how careful design decisions make temporal climate data much more interpretable.
